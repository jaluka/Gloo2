Memory Skills For Business Professionals
7 Day Follow-Up Program

Day 6

Memory Applications
- New Journey

REVIEW - Yesterday you created a second journey. Take a moment now and see how quickly you can run through your new journey from start to finish.

Then run through it backwards too, as quickly as you can.

And as you are now approaching the end of this short follow-up program, time to put things to the test and prove to yourself that you can remember anything you choose to.

Memory Challenge = The Periodic Table

Use your house journey to memorise the first 20 elements of the periodic table, as quickly as you can…

Hydrogen
Helium
Lithium
Beryllium
Boron
Carbon
Nitrogen
Oxygen
Fluorine
Neon
Sodium
Magnesium
Aluminium
Silicon
Phosphorus
Sulfur
Chlorine
Argon
Potassium
Calcium

Simply focus on each element, create a visual prompt for it then connect it to the appropriate stop on your journey.

Once you've done all 20, close your eyes and mentally run through your journey - BACKWARDS! - reciting the 20 elements as you go.

If you prefer to use something that's business-related that's fine too, your choice. The main purpose of this exercise is to show you that with the skills you now have you can remember anything you choose to - even chemistry!

So give it a go!

Memory Applications
- Names

REVIEW - Yesterday you created visual reminders for 5 female names beginning with 'B'. What was your image for…

Brenda?
Belinda?
Beth?
Becky?
Bianca?

Open up your Visual Vocabulary files and review the visual reminders you created. Also briefly review the 10 prompts you created for 'A' names.

Now think of an image for the following 5 male names beginning with 'B'…

Bob
Benny
Brad
Brian
Bruce

Record them in your Visual Vocabulary file for male names. Add a location for each name if this helps. Then close your eyes and picture your images for the 5 names once more.

End of Day 6!
