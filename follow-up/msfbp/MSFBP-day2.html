Memory Skills For Business Professionals
7 Day Follow-Up Program

Day 2

Key Memory Skills
- Visualisation and Linking

REVIEW - Yesterday you linked together 3 pairs of unrelated words. Do you recall what you connected to...

Birthday cake?
Banana?
Denist?

If you can correctly recall the answers, well done!

If not, please go back to Day 1 to see the answers, then strengthen your connections by adding more action, perhaps some emotion or even put yourself into the picture.

Memory Applications
- Names

REVIEW - Yesterday you created visual reminders for the first names (or middle names) of the other people on the workshop. Mentally review your images and see how many you can recall.

If you need to, open up your Visual Vocabulary files and review the chosen images you filled in. Now think of an image for the following 5 female names beginning with 'A'…

Alexis
Amanda
Annette
Angela
Annie

Record them in your Visual Vocabulary file for female names. Add a location for each name if you find this helpful. When you've done this, close your eyes and picture your images for the 5 names once more.

End of Day 2!
